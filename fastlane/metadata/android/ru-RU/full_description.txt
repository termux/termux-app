Termux - это приложение эмулятора терминала, с большим набором утилит командных строки, перенесённых на Android.
Главная цель это предоставить командную строку Linux пользователям мобильных устройств без получения прав от root и другой специальной настройки.

<ul>
<li>Командные оболочки Bash и Zsh.</li>
<li>Редактирование файлов с помощью Nano и Vim.</li>
<li>Доступ к серверам по SSH.</li>
<li>Компилирование C/C++ с Clang.</li>
<li>Используйте консоль Python как карманный калькулятор.</li>
<li>Работать с Git.</li>
<li>Запускать текстовые игры с Frotz.</li>
</ul>

Сначала настраивается небольшая базовая система. GNU Bash, Coreutils, Findutils и другие основные утилиты доступны из коробки.
Дополнительно, доступно более тысячи других пакетов для установки через утилиту <em>pkg</em>, которая в настоящее время является обвёрткой для менеджера пакетов <em>apt</em>.
Всё предоставленное программное обеспечение было исправлено и составлено с Android NDK, чтобы обеспечить максимальную совместимость с ОС Android.

Чтобы найти больше советов по использованию приложения, нажмите на любое место в терминале и выберите опцию меню «Справка» для доступа к <a href="https://wiki.termux.com/wiki/Main_Page>Вики Termux</a>.
