OpenClaw Assistant is a powerful personal AI assistant and terminal environment for Android. It integrates the OpenClaw assistant gateway and Ollama local models directly into a high-performance Linux environment.

* Run local AI models like phi4-mini with Ollama.
* Manage your personal AI gateway with a dedicated dashboard.
* Enjoy the Bash and Zsh shells.
* Edit files with nano and vim.
* Access servers over SSH.
* Compile code and use Python, Git, and more.

OpenClaw Assistant provides a dedicated AI Dashboard for easy management of local services. It automatically handles the setup of OpenClaw and provides an embedded web interface for a seamless assistant experience.

Built on the foundation of the Termux project, it maintains full compatibility with a vast library of Linux packages while adding first-class support for modern AI workflows.
